## 追加したFAQの詳細

### 基本（3問）

1. **uploaderとrsync/scpとの違いは？**
   - uploaderの優位性を明確に説明
   - Git差分自動検出、YAML設定管理、視覚的差分確認などを強調

2. **どのプロトコルを選べばいいですか？**
   - rsync/sftp/scp/localの選択基準を提示
   - 用途別の推奨プロトコルを明示

3. **updateとmirrorの違いは？**
   - 同期モードの違いを明確に説明
   - mirrorモードの危険性を警告（自動削除）
   - dry-runでの確認を推奨

### インストール・設定（3問）

4. **Denoのインストールが必要ですか？**
   - Denoが必要であることを明示
   - macOS/Linux/Windowsのインストール方法を提示

5. **設定ファイルはどこに置けばいいですか？**
   - uploader.yamlの配置場所を明示
   - 設定ファイルの検索順序を説明

6. **環境変数を使えますか？**
   - ${VAR_NAME}形式の使用方法を提示
   - .envファイルまたはシェルでの設定例を提示

### ファイル除外（2問）

7. **特定のファイルを除外したい**
   - _global.ignore_groupsの使い方を説明
   - 基本的な除外パターンの例を提示

8. **ターゲットごとに異なる除外パターンを使いたい**
   - ターゲット固有のignore設定を説明
   - use/addオプションの使い分けを提示

### トラブルシューティング（3問）

9. **「Permission denied」エラーが出る**
   - SSH鍵のパーミッション確認方法
   - リモートディレクトリの書き込み権限確認
   - rsync_path: "sudo rsync"の設定

10. **「Connection refused」エラーが出る**
    - SSH接続の確認方法
    - ホスト名・ポート番号・ファイアウォールの確認

11. **ファイルが予期せず削除された**
    - mirrorモードの危険性を再度警告
    - 対策として--dry-run、バックアップ、updateモードを提示

### 高度な使い方（5問）

12. **複数のサーバーに同時にアップロードしたい**
    - defaultsとtargetsの使い方を説明
    - 複数ターゲットの設定例を提示

13. **ブランチ以外の差分をアップロードしたい**
    - コミットハッシュやタグの指定方法を説明

14. **特定のファイル・ディレクトリだけアップロードしたい**
    - fileモードの使い方を説明
    - srcでのファイル指定例を提示

15. **dry-runの結果をファイルに保存したい**
    - リダイレクトを使った方法を提示

16. **ログファイルはどこに保存されますか？**
    - デフォルトではログファイルが作成されないことを明示
    - --log-fileオプションの使い方を説明

### その他（3問）

17. **プロファイル一覧を確認したい**
    - --listオプションの使い方を説明
    - 出力例を提示

18. **バグを見つけた・機能要望がある**
    - GitHub IssuesとDiscussionsへの誘導

19. **まだ解決しない場合**
